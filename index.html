⧡ Quick Start Guide
ApexIQ v17 is a self-contained global AI business intelligence platform running entirely in a single HTML file — no server, no installation, no external dependencies beyond the AI engine of your choice.
What You Need
One free API key is all you need to activate the full platform. Get one from any of these sources at zero cost:
• Google Gemini 2.5 Flash — aistudio.google.com (free tier, generous limits)
• Groq — console.groq.com (free, very fast)
• GitHub Models — github.com/marketplace/models (free with GitHub account)
First 5 Minutes
1
Open Settings
Click Settings & Engines at the bottom of the left sidebar. Find your engine (e.g. Google Gemini), paste your API key, and click 💾 Save Settings.
2
Set Session Context
Click 📌 Set Context on the Command Bridge. Enter your company name, industry, and target market. This context is automatically injected into every module prompt — once per session is all you need.
3
Run Your First Module
Navigate to Market Intelligence. Type a research question, select your region and sector, and click ▶ Scan Market. Your first AI-powered report appears in seconds.
4
Chain to the Next Module
At the bottom of any output, find the Chain → buttons. Click one to pre-load that output as context in the next module and navigate there automatically.
5
Use the Co-Pilot
Click APEX Co-Pilot in the sidebar. Describe your business goal in plain language. Co-Pilot recommends the exact module chain, explains what data transfers at each step, and can execute the entire chain autonomously.
⚠ Privacy Note: ApexIQ never transmits your data anywhere except directly to the AI engine you configure. No telemetry, no analytics, no third-party services. Open-Meteo (weather) and OpenStreetMap (POI) use public free APIs with city-name-only queries.
AI-Powered Manual
Every other section of this manual is generated on demand by your configured AI engine — personalised to your session context, cached for the session so the API is called only once per section. Navigate to any section to generate it now.
